🌐 Deploy Lex AI Globally

How to launch this chatbot for law students worldwide — 4 easy methods, all free.

1 Add Your Anthropic API Key

Open this HTML file in any text editor. Find the sendMessage() function. The API call already uses the correct endpoint. You only need to add your key to the request header:

'x-api-key': 'sk-ant-YOUR_KEY_HERE',
'anthropic-version': '2023-06-01',
'anthropic-dangerous-direct-browser-calls': 'true'

⚠ For production, always proxy the API call through your backend so the key is never exposed in browser source.

2 Free Hosting Options (No Code Needed)

Upload the single HTML file to any of these free platforms:

GitHub Pages Netlify Drop Vercel Cloudflare Pages Render Tiiny.host

For GitHub Pages: create a repo → upload the file as index.html → enable GitHub Pages in Settings → your URL is live instantly, globally.

3 Production-Safe Backend Proxy (Recommended)

To protect your API key from public exposure, create a tiny serverless function:

// Netlify / Vercel Edge Function (Node.js)
export default async (req) => {
const body = await req.json();
const r = await fetch('https://api.anthropic.com/v1/messages', {
method: 'POST',
headers: {
'x-api-key': process.env.ANTHROPIC_KEY,
'anthropic-version': '2023-06-01',
'Content-Type': 'application/json'
},
body: JSON.stringify(body)
});
return new Response(await r.text(), {status: r.status});
};

Then change the fetch URL in this file from https://api.anthropic.com/v1/messages to your proxy URL.

4 Customise for Your Law School or Region

The jurisdiction dropdown already supports 16 countries. To add more, edit the <select id="jurisdictionSelect"> element and extend the jurisdictions object in the script. The AI system prompt auto-adapts to whichever jurisdiction is selected.
